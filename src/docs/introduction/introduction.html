Create custom component with docs

  1. download initial component template structure from vuestrap-component-template github page
  2. run npm install
  3. run npm run docs
  4. navigate to yout browser http://localhost:8080
  5. add your component (logic, html and scss) in src/components/componentname folder
  6. add demo page in src/docs/democomponentname folder with additional route info
  7. Update introduction page in src/docs/introduction/introduction.html with your installation and examples details
  8. Update package.json file
  9. build package with npm run dist
  10. create repo on github and commit your code base
  11. create gh-pages branch and push your code there as well
  12. publish your component to NPM npm publish
  13. publish your component to Bower bower register componentName git://github.com/username/componentName.git
  14. (optional) create a pull request on vuestrap page and add a link to your newly created gh-pages demo page

If you done everything correctly, you should now see your docs live at http://username.github.io/reponame.

Note: vuestrap-component-template includes everything you need to kick start your development. Remember to check the sample 'Hello World' component and read some comments before you start :)
